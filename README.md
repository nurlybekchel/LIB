# 📚 Онлайн-библиотека

Веб-приложение для онлайн-чтения книг с системой авторизации и админ-панелью.

## 🖥️ Превью

| Главная | Каталог | Профиль | Админ-панель |
|---------|---------|---------|--------------|
| Список книг на главной | Поиск и фильтрация по жанру | История чтения | Управление книгами и пользователями |

## ✨ Функционал

### Для читателей
- 📖 Чтение книг онлайн прямо в браузере (PDF)
- 🔍 Поиск книг по названию и автору
- 🎭 Фильтрация по жанрам
- 👤 Страница профиля с историей прочитанных книг
- 📊 Статистика чтения

### Для администраторов
- ➕ Добавление новых книг с загрузкой PDF
- ✏️ Редактирование информации о книгах
- 🗑️ Удаление книг
- 👥 Управление пользователями (роли, удаление)
- 📊 Дашборд с графиками и статистикой

## 🛠️ Технологии

**Backend**
- Node.js + Express
- SQLite (better-sqlite3)
- JWT авторизация
- bcryptjs для хеширования паролей
- Multer для загрузки PDF файлов

**Frontend**
- Vanilla JavaScript
- CSS переменные (темная тема)
- Chart.js для графиков на дашборде

## 🚀 Установка и запуск

### 1. Клонируй репозиторий

```bash
git clone https://github.com/ТВО_ИМЯ/library.git
cd library
```

### 2. Установи зависимости

```bash
npm install
```

### 3. Запусти сервер

```bash
node backend/server.js
```

Или с авто-перезагрузкой:

```bash
npx nodemon backend/server.js
```

### 4. Открой в браузере
http://localhost:3000

## 👤 Первый запуск

При первом запуске база данных заполняется тестовыми книгами автоматически.

**Первый зарегистрированный пользователь становится администратором.**

Зарегистрируйся на `/` → ты получишь роль `admin` и доступ к `/admin.html`

## 🔐 API эндпоинты

### Книги
| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/books` | Список всех книг |
| GET | `/api/books/:id` | Одна книга |
| POST | `/api/books` | Добавить книгу (admin) |
| PUT | `/api/books/:id` | Редактировать книгу (admin) |
| DELETE | `/api/books/:id` | Удалить книгу (admin) |
| POST | `/api/books/:id/pdf` | Загрузить PDF (admin) |
| GET | `/api/books/meta/genres` | Список жанров |

### Авторизация
| Метод | URL | Описание |
|-------|-----|----------|
| POST | `/api/auth/register` | Регистрация |
| POST | `/api/auth/login` | Вход |
| GET | `/api/auth/me` | Текущий пользователь |

### Админ
| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/admin/stats` | Статистика |
| GET | `/api/admin/users` | Список пользователей |
| PATCH | `/api/admin/users/:id/role` | Изменить роль |
| DELETE | `/api/admin/users/:id` | Удалить пользователя |

## ⚙️ Переменные окружения

Создай файл `.env` в корне (опционально):

```env
JWT_SECRET=твой_секретный_ключ
PORT=3000
```

По умолчанию используется `library_secret_key_2024` — **смени в продакшене**.
